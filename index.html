Project description:

Primary Objective: Develop an IOS app for museum visitors.

This apps is supposed to give the users an enhanced experience when they visit an museum. The app should be able to detect where the user is, and show them relative information regarding there current location within the premises. For instance, lets say a user is near a painting, and the painting does have enough information on display, the app should display relevant information about the painting. If the infomarion provided not sufficient, the user should be able to ask question, and get relevant answers from within the app itself.

For the admins of the museum the app should be able to give analytics. Such as on what area are the users most interested in, what area of the museum are the users asking the most questions. ETC...

Secondary Objective: Develop an Android app with same Functionality.
